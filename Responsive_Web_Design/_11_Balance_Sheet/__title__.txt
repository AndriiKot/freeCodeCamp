Learn More About CSS Pseudo Selectors by Building 
A Balance Sheet

You can use CSS pseudo selectors 
to change specific HTML elements.

In this course, you'll build a balance sheet using pseudo selectors.
You'll learn how to change the style of 
an element when you hover over it with your mouse,
and trigger other events on your webpage.

RU:

Балансовый отчет

Вы можете использовать псевдоселекторы CSS
для изменения определенных элементов HTML.

В этом курсе вы построите баланс, используя псевдоселекторы.

и запускать другие события на вашей веб-странице.
