Data can be stored and accessed in many ways.

You already know some common JavaScript data structures — arrays and objects.

In this Basic Data Structures course, you'll 
learn more about the differences between 
arrays and objects, and which to use in different situations. 
You'll also learn how to use helpful 
JS methods like splice() and Object.keys() to access and manipulate data.

RU:

Вы уже знакомы с некоторыми распространенными структурами данных JavaScript — массивами и объектами.

В этом курсе «Основные структуры данных» вы узнаете больше о различиях между массивами и объектами и о том, 
что использовать в различных ситуациях. Вы также узнаете, как использовать полезные методы JS, 
такие как splice()и Object.keys()для доступа к данным и манипулирования ими.
